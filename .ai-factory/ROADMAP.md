# Project Roadmap: Slovor Marketplace

> Premium Slovakian classifieds marketplace with "Customer CRM" premium UI design.

## Milestones

### 🏗️ Foundation & Core
- [x] **Avant-Garde Design System** — Unified "Pro Max" UI tokens and components.
- [x] **Supabase Infrastructure** — Table schemas, RLS policies, and Auth integration.
- [x] **Multi-language Framework** — i18n support for EN, SK, RU, CS.
- [x] **Global Style Unification** — Replaced legacy styles with DS standards.

### 🏠 Listing Discovery
- [x] **Category Management** — Dynamic category listing and discovery.
- [x] **Listing Detail & Search** — Search flow and detailed listing view.
- [x] **Smart Search Interactions** — Refining filter logic and search UX.

### 👤 User & Seller Experience
- [x] **Authentication Flow** — Secure login/register with Supabase.
- [x] **Premium User Dashboard** — High-end CRM-style dashboard for listing management.
- [ ] **Seller Profile Enhancements** — Advanced seller statistics and verification.

### 🛠️ Advanced Features
- [ ] **Advanced Media Management** — Multi-file uploads and image optimization.
- [ ] **Messaging System** — In-app communication between buyers and sellers.
- [ ] **Payment Integration** — Stripe/Braintree for premium listing promotions.
