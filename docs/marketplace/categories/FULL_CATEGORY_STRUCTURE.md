# Оптимальная структура категорий для маркетплейса объявлений Словакии/Чехии

## Анализ существующих платформ

### Основные платформы для анализа:
- **Bazos.cz** / **Bazos.sk** - 20+ главных категорий с 3-5 уровнями глубины
- **OLX.pl** - аналогичная структура
- **Jofogas.hu** - работающая в Венгрии

### Общие выводы:
1. Все платформы используют **2-3 уровня категоризации** (макро → микро)
2. Ключевые категории почти идентичны всем маркетплейсам
3. Рекомендуется **20-25 главных категорий** (не более 30)
4. Каждая главная категория может содержать **5-20 подкатегорий**

---

## 🎯 Оптимальная структура для твоего маркетплейса

### Структура: 3 уровня (МАКРО → СРЕДНИЙ → МИКРО)

```
ГЛАВНАЯ КАТЕГОРИЯ
  └─ ПОДКАТЕГОРИЯ
      └─ КОНКРЕТНЫЙ ТИП (опционально)
```

## 🔐 РЕКОМЕНДАЦИИ ПО ВНЕДРЕНИЮ

### Фаза 1: Базовая структура (MVP)
1. Импортировать 25 главных категорий
2. По 3-5 подкатегорий в каждой
3. Не более 3-х уровней глубины

### Фаза 2: Оптимизация
1. Анализировать реальное распределение объявлений
2. Добавлять микро-категории по спросу
3. Удалять неиспользуемые категории

### Фаза 3: Расширение
1. Добавлять кросс-категорийный поиск
2. Внедрить теги и атрибуты
3. ML-классификация объявлений

- Гарантия
