# Резюме документации Slovor Marketplace

## Обзор проекта

Slovor Marketplace - премиальный классифайд маркетплейс для Словакии, построенный на Next.js 15, React 19, Supabase и Tailwind CSS. Проект находится в фазе бета-разработки, готов к инвестициям, с уникальным "avant-garde" дизайном как ключевым преимуществом.

## Архитектура системы (ARCHITECTURE.md)

- **Фронтенд:** Next.js 15 с App Router, React 19, TypeScript (строгий режим)
- **Бэкенд:** Supabase (Auth, PostgreSQL, Storage) с Row Level Security
- **UI:** Tailwind CSS, Framer Motion, Radix UI
- **Развертывание:** Vercel
- **Архитектура:** Атомарный дизайн компонентов, серверные компоненты, ISR для статических страниц
- **Безопасность:** RLS политики, валидация входных данных, защита от XSS/SQL-инъекций

## Принципы разработки (PRINCIPLES.md)

**Обязательные стандарты:**

- Дизайн-первый подход (UX превыше всего)
- Строгий TypeScript (без `any`, явные типы)
- Качество превыше скорости разработки
- Атомарный дизайн компонентов
- Производительность как функция
- Тестирование (80% покрытие минимум)
- Документация кода с JSDoc
- Conventional Commits

## Quick Start (QUICK_START.md)

Быстрая настройка за 5 минут:

1. Клонировать репозиторий
2. Перейти на ветку dev
3. `npm install`
4. Настроить .env.local с Supabase
5. `npm run db:reset`
6. `npm run dev`

## Руководство разработчика (DEVELOPMENT.md)

- Ветки: main (прод), dev (разработка), feature/\*
- Workflow: dev → feature branches → PR → merge
- Тестирование: Vitest, 80% покрытие
- База данных: Миграции в supabase/migrations/
- Развертывание: Vercel, автоматическое из main/dev

## Структура веток

- `main`: Продакшн
- `dev`: Активная разработка
- `docs`: Документация (текущая ветка)
