# Design Master Protocol

> **TRIGGER**: Any task involving UI, CSS, Components, Layout, or Aesthetics.

## 🛑 STOP & READ

You must follow the **Design Protocol** located at:

👉 **`.design-protocol/README.md`**

## Summary of Protocol

1.  **Constraints First**: Check `.design-protocol/core/global-constraints.md`.
2.  **Theme Alignment**: Apply the active theme rules from `.design-protocol/themes/`.
3.  **Execute**: Implement changes using the defined palette and physics.
4.  **Premium Check**: Does it look Premium? If it looks "basic" or "bootstrap", it is failed. Follow `MEMORY.md` rules (Solid, Clean, Data-Dense).
