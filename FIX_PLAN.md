## 🎯 Анализ "Проёбов" (Причины сбоев)
1. **Разрыв связи WSL**: Использование `pkill -9 -f node` убивает не только сервер Next.js, но и мост связи Agent-IDE. **РЕШЕНИЕ**: Использовать точечный `kill` только для порта 3000.
2. **Фрагментация Env**: Наличие `env.ts` и `env.mjs` приводило к рассинхрону ключей. **СДЕЛАНО**: Все объединено в один файл `lib/env.ts`, зависимости обновлены.
3. **Invalid API Key**: Вероятно, был вызван тем, что Middleware использовал старый метод загрузки переменных. **СДЕЛАНО**: Middleware синхронизирован с новым `lib/env.ts`.
