# slovor-mp

Slovor Marketplace is a customized platform enabling translation management, order tracking, and client-translator collaboration for localization services.

---

## 🚀 Key Features
- User authentication and role management (client/translator)
- Document upload and translation progress tracking
- Billing and payment integration
- Automated notification templates

---

## 🛠️ Tech Stack & Setup
- **Core Environment**: Setup instructions depend on the stack.
- Make sure to create a `.env` file for local development configs (do not push keys to git).
